# Github 관련 수업

1. C드라이브에 여러분영어이름_temp 라는 새폴더 생성
2. 폴더 우클릭 -> git bash -> code .
3. 터미널 켜서 git init
4. git config user.name "여러분깃허브아이디"
5. git config user.email "여러분깃허브이메일"
